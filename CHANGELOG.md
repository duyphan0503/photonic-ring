All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-01-23

### Added

- Initial release of Photonic Ring AI Texture Generator.
- High-contrast Height map generation (Guided Filter + CLAHE).
- Variance-based Roughness map generation.
- Improved Normal map quality (Scharr operator).
- Fully integrated Godot Editor UI.
- Multi-threaded Rust core for performance.
- Professional documentation and PBR samples.
